# Códigos

Esta carpeta contiene los programas desarrollados para las diferentes actividades del laboratorio del robot Phantom X Pincher.

| Archivo | Actividad |
|---------|-----------|
| `move_turtle.py` | **Actividad 7.** Movimiento simultáneo del robot mediante ROS2. |
| `fk_phantom.py` | **Actividad 11.** Implementación de la cinemática directa del Phantom X Pincher. |
| `Actividad_12.py` | **Actividad 12.** Implementación de la cinemática inversa, selección de soluciones válidas y ejecución del movimiento en RViz. |
| `teach_repeat.py` | **Actividad 13.** Implementación del modo de enseñanza y repetición de poses utilizando almacenamiento en un archivo YAML. |

Cada archivo puede ejecutarse de forma independiente.
